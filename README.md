# WindowTop窗口置顶工具
&emsp;&emsp;一个Windows平台的窗口置顶工具。目前添加了以下功能：   

1. 置顶窗口功能；
2. 窗口穿透功能；
3. 窗口透明度设置功能；
4. 窗口自定义画中画功能；
5. 简易笔记的功能；
6. 白名单功能；
7. 获取窗口输入法状态功能。
